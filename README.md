### 原理

# Unity_Helper_BehaviourTree

*C# 行为树*

*Unity 可放心食用*

### 包含基础节点
Action
Behavior
Composite
Condition
Decoreator
Filter
Monitor
Parallel
Selector
Sequence

### Action 节点
* ActionLog
* ActionWait - 等待x秒后，返回Succeed，过程中持续Running

### Condition 节点
* ConditionTrue
* ConditionFalse

### DecoratorNodes
* DecoratorRepeat - 装饰器节点：多次执行
* DecoratorInvert - 装饰器节点：取反

## BTreeBlackBoard 黑板
可记录环境信息、树内共享信息
* EnableRunningLog - 打印运行时log，便于跟踪运行情况
